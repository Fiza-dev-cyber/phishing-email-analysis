Mini project on phishing email detection and analysis
# Phishing Email Analysis

## Objective
To analyze phishing emails and understand common attack patterns used to deceive users.

## Tools Used
- VirusTotal
- Email Header Analyzer
- Browser-based analysis

## Methodology
Collected phishing email samples and examined sender details, URLs, language patterns, and urgency tactics.

## Findings
Identified common red flags such as spoofed domains, shortened URLs, and fear-based messaging.

## Conclusion
Phishing attacks rely more on psychological manipulation than technical complexity. Awareness and training are key defenses.
